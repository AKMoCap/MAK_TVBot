Alert Message Template (Minimal)

Paste this into TradingView's "Message" field

{
    "secret": "paste-your-actual-secret-here",
    "action": "buy",
    "coin": "BTC",
    "indicator": "your-indicator-key"
}
Coin Config Defaults: Leverage, collateral, stop loss, and take profit will automatically use your Coin Configuration defaults if not specified in the webhook.
Field Reference
FieldRequiredDescription
secret Yes The Webhook Secret from your indicator above (unique per indicator)
action Yes "buy" or "sell" - hardcode for indicator alerts
coin Yes Coin symbol exactly as shown in Hyperliquid (e.g., "BTC", "ETH")
indicator Recommended Your Webhook Key from above - links trades to this indicator for tracking
leverage Optional Override leverage (uses Coin Config default if omitted)
collateral_usd Optional Override collateral amount (uses Coin Config default if omitted)
stop_loss_pct Optional Override stop loss % (uses Coin Config default if omitted)
close_position Optional Set to true to close existing position for this coin
Important: Create separate alerts for buy and sell signals since action must be hardcoded for indicator-based alerts.
Pro Tip: Start with TESTNET enabled to verify your setup works correctly before trading with real funds.
Sample Pine Scripts
//@version=5
strategy("RSI Strategy for MAK Bot", overlay=true)

// Parameters
rsiLength = input.int(14, "RSI Length")
oversold = input.int(30, "Oversold Level")
overbought = input.int(70, "Overbought Level")

// Calculate RSI
rsiValue = ta.rsi(close, rsiLength)

// Generate signals
longCondition = ta.crossover(rsiValue, oversold)
shortCondition = ta.crossunder(rsiValue, overbought)

if (longCondition)
    strategy.entry("Long", strategy.long)
    alert('{"secret":"YOUR_SECRET","action":"buy","coin":"' + syminfo.ticker + '","leverage":10,"collateral_usd":100,"indicator":"rsi-oversold"}', alert.freq_once_per_bar)

if (shortCondition)
    strategy.entry("Short", strategy.short)
    alert('{"secret":"YOUR_SECRET","action":"sell","coin":"' + syminfo.ticker + '","leverage":10,"collateral_usd":100,"indicator":"rsi-overbought"}', alert.freq_once_per_bar)
//@version=5
strategy("EMA Crossover for MAK Bot", overlay=true)

// Parameters
fastLength = input.int(9, "Fast EMA")
slowLength = input.int(21, "Slow EMA")

// Calculate EMAs
fastEMA = ta.ema(close, fastLength)
slowEMA = ta.ema(close, slowLength)

// Plot EMAs
plot(fastEMA, color=color.blue, title="Fast EMA")
plot(slowEMA, color=color.red, title="Slow EMA")

// Generate signals
longCondition = ta.crossover(fastEMA, slowEMA)
shortCondition = ta.crossunder(fastEMA, slowEMA)

if (longCondition)
    strategy.entry("Long", strategy.long)
    alert('{"secret":"YOUR_SECRET","action":"buy","coin":"' + syminfo.ticker + '","leverage":10,"collateral_usd":100,"indicator":"ema-cross"}', alert.freq_once_per_bar)

if (shortCondition)
    strategy.entry("Short", strategy.short)
    alert('{"secret":"YOUR_SECRET","action":"sell","coin":"' + syminfo.ticker + '","leverage":10,"collateral_usd":100,"indicator":"ema-cross"}', alert.freq_once_per_bar)
//@version=5
strategy("MACD Strategy for MAK Bot", overlay=false)

// Parameters
fastLength = input.int(12, "Fast Length")
slowLength = input.int(26, "Slow Length")
signalLength = input.int(9, "Signal Length")

// Calculate MACD
[macdLine, signalLine, histLine] = ta.macd(close, fastLength, slowLength, signalLength)

// Plot
plot(macdLine, color=color.blue, title="MACD")
plot(signalLine, color=color.orange, title="Signal")
plot(histLine, style=plot.style_histogram, color=histLine > 0 ? color.green : color.red, title="Histogram")

// Generate signals
longCondition = ta.crossover(macdLine, signalLine) and macdLine < 0
shortCondition = ta.crossunder(macdLine, signalLine) and macdLine > 0

if (longCondition)
    strategy.entry("Long", strategy.long)
    alert('{"secret":"YOUR_SECRET","action":"buy","coin":"' + syminfo.ticker + '","leverage":10,"collateral_usd":100,"indicator":"macd"}', alert.freq_once_per_bar)

if (shortCondition)
    strategy.entry("Short", strategy.short)
    alert('{"secret":"YOUR_SECRET","action":"sell","coin":"' + syminfo.ticker + '","leverage":10,"collateral_usd":100,"indicator":"macd"}', alert.freq_once_per_bar)
